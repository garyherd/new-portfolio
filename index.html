Frogger Clone

Using the given visual assets and a game loop engine, the goal was to add a number of entities to the game including the player characters and enemies to recreate the classic arcade game Frogger. Games have a lot of objects and those objects do a lot of different things; but sometimes they do some very similar things as well. This was a great opportunity to practice object-oriented programming, an important programming paradigm that influences an application architecture and provides performance optimization. I have plenty to learn about object-oriented programming, and what I learned in this project was how to use the object prototype to create an object constructor function.

  • Client: Udacity Front End Nanodegree
  • Date: June 2016
  • Links: Website ; Source Code

ISO Prices

This page scrapes data from the four major electric wholesale markets: ERCOT, PJM, NYISO, and ISO-NE. The backend was written in Python, and using the Beautiful Soup library. The front-end uses a free Bootstrap theme from Bootswatch. The scraping part was done to help the developers at my company with a project they were working on.

Neighborhood Map

This single-page application uses the Google Maps API and the Foursquare API to display information around my local area. The user can filter the locations by several types, and the map displays markers corresponding to the selected locations. It also uses the lightweight framework W3.CSS. The documentation for the Google Maps API is excellent, and the Foursquare is pretty good too.
